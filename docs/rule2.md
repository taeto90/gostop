1. 뻑 (설사 / 똥)
가장 허탈하면서도 판의 긴장감을 높이는 규칙입니다.

상황: 바닥에 있는 패를 먹기 위해 패를 냈는데, 더미에서 뒤집은 패마저 같은 월(달)의 패가 나와서 바닥에 3장이 겹치게 되는 경우입니다.

결과: 패를 가져오지 못하고 그대로 바닥에 쌓아둬야 합니다.

자뻑: 만약 내가 싼 뻑을 나중에 내가 직접 먹게 되면, '자뻑'이라 하여 상대방들로부터 피를 2장씩 가져옵니다. (첫 턴에 뻑을 하면 '첫뻑'이라 하여 보너스 점수를 주는 지역도 있습니다.)

2. 폭탄
손에 패가 많을 때 단숨에 판세를 뒤집는 기술입니다.

상황: 바닥에 같은 월의 패가 1장 깔려 있고, 내 손에 같은 월의 패 3장이 모두 있을 때, 이 3장을 한꺼번에 내리치며 먹는 경우입니다.

결과:

상대방의 피를 1장씩 뺏어옵니다.

승리 시 점수가 2배(흔들기 효과)가 됩니다.

패 3장을 한 번에 냈으므로 남들보다 패가 2장 부족해집니다. 이를 채우기 위해 남은 턴 중 두 번은 패를 내지 않고 더미에서 패만 뒤집을 수 있는 권리가 생깁니다.

3. 흔들기
패가 좋지 않아도 점수를 뻥튀기할 수 있는 전략입니다.

상황: 내 손에 같은 월의 패가 3장 있을 때, 이를 다른 사람들에게 보여주며 "흔든다"고 선언하는 것입니다. (폭탄을 쓸 수 없는 상황—바닥에 같은 월의 패가 없을 때 주로 사용합니다.)

결과: 해당 판에서 내가 승리할 경우, 최종 점수를 2배로 계산합니다. 만약 두 번 흔들면 4배가 됩니다.

4. 쪽 (뽀뽀)
우연이 가져다주는 기분 좋은 보너스입니다.

상황: 바닥에 내가 먹을 수 있는 패가 없어서 아무 패나 냈는데, 더미에서 뒤집은 패가 방금 내가 낸 패와 같은 월인 경우입니다.

결과: 우연히 패를 먹게 된 보상으로 상대방의 피를 1장씩 뺏어옵니다. (단, 쪽으로 패를 먹은 직후 바닥에 패가 하나도 남지 않으면 아래의 '싹쓸이'와 중복 적용되어 피를 2장 뺏어옵니다.)

5. 싹쓸이 (쓸)
판을 깨끗하게 청소하고 보너스를 챙기는 규칙입니다.

상황: 내가 패를 내고 더미를 뒤집어서 패를 가져간 뒤, 바닥에 남아있는 패가 단 하나도 없을 때입니다.

결과: "쓸!"이라고 외치며 상대방의 피를 1장씩 뺏어옵니다.

예외: 마지막 턴(각자 패가 1장씩 남았을 때)에 발생하는 싹쓸이는 보통 피를 뺏어오지 않는 것이 관례입니다.


상황,뺏어오는 피 개수,비고
쪽 / 쓸 / 따닥 / 폭탄,1장,가장 일반적인 보너스
자뻑,2장,자기가 싼 똥을 치웠을 때
쪽 + 쓸,2장,쪽을 해서 바닥이 다 비었을 때
