## ⚪ 부분/단순 구현 (추후 옵션)

### 5. 분배 시각화 (4장씩 / 3장씩 vs 5장씩)
- **현재**: 라운드 기반 단순 분배 (결과 동일)
- **누락**: 정통 분배 — 3인 4장씩 한 바퀴 → 바닥 3장 → 3장씩 → 바닥 3장
- **이유**: 시각적 분배만 다름, 결과 카드 분포 동일
- **상태**: 게임플레이 영향 X. 시각 애니메이션만 추가 가능 (1일)

### 6. 첫뻑 보너스 점수
- **상태**: 시각효과(⭐ 첫뻑!) 표시는 OK
- **누락**: rule3 "미리 정한 금액" 점수 보너스 — 친구용 MVP는 돈 X라 미적용
- **상태**: 옵션 추가 가능 (방장 룰 모달에 토글)
