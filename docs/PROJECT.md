# 프로젝트 소개 (Project Overview)

## 한 줄 요약

> 한국 고스톱(화투) 게임을 친구들과 화상채팅으로 즐길 수 있는 웹 애플리케이션

## 프로젝트 배경

### 만드는 이유
- 친구들끼리 모이지 않아도 고스톱을 함께 즐길 수 있는 환경
- 단순한 게임이 아닌, 친구놀이의 분위기를 살리는 화상/음성 통합
- 관전자도 함께 떠들 수 있는 공간 (5인 동시 참여)

### 핵심 차별점
- **이모지 반응**: 영상 위에 떠오르는 즉각적 감정 표현 (5명이 동시에 떠들면 정신없으니까)
- **게임 통계 보드**: "오늘의 호구", "오늘의 광부" 등 친구놀이 추억 거리
- **관전자 사이드 베팅**: 게임 안 해도 같이 즐길 수 있는 참여 요소

## 목표 (Goals)

### 단기 목표 (MVP)
- ✅ 친구 5명이 동시 접속해 화상채팅하며 게임 진행
- ✅ 2-3명이 게임 + 1-2명 관전 (또는 모두 관전 가능)
- ✅ 표준 고스톱 룰 + 보너스 룰 (흔들기/폭탄/박/고스톱)
- ✅ URL 공유로 즉시 입장 (회원가입 없이)

### 중기 목표 (배포 후)
- ✅ 20명 동시 접속, 4방 × 5명 시나리오 안정 운영
- ✅ 모바일 브라우저 지원 (반응형)
- ✅ 게임 통계/리플레이로 친구놀이 즐길 거리 확장

### 비목표 (Non-Goals)
- ❌ **상용 서비스**: 한국 게임산업법상 사행성 게임 규제 회피
- ❌ **현금/현금성 재화 베팅**: 절대 불가 (법적으로 차단)
- ❌ **신규 회원가입/본인인증**: 친구용이므로 닉네임만으로 입장
- ❌ **광고 수익 모델**: MVP 단계에선 친구 그룹 내 사용만 상정
- ❌ **앱 스토어 출시**: 웹앱으로만 운영

## 타겟 사용자

### 핵심 시나리오
**5명의 친구가 주말 저녁에 모여서**
- 호스트가 방을 만들고 URL을 카톡으로 공유
- 친구들이 링크 클릭 → 닉네임 + 이모지 아바타 입력 → 즉시 입장
- 3명은 게임에 참여, 2명은 관전 (다음 판에 교대 가능)
- 화상채팅으로 떠들면서 1-2시간 즐기다가 자연스럽게 종료

### 사용자 가정
- 화투 룰을 어느 정도 알거나 배우고 싶음
- 친구들과 같이 떠들고 싶지만 만나기 어려움
- 카카오톡 보이스톡/페이스타임 같은 기존 도구로는 게임을 같이 못해서 불편함을 느낌
- 컴퓨터 또는 스마트폰을 평범하게 다룸

## 게임 룰 요약

### 화투 48장 구성
- **광** 5장 (1·3·8·11·12월)
- **열끗** 9장 (2·4·5·6·7·8·9·10·12월)
- **띠** 10장 (홍단 3 + 청단 3 + 초단 3 + 비띠 1)
- **피** 24장 (일반 21 + 쌍피 3)
- **고도리** 마커: 2·4·8월 열끗
- **비광**: 12월 광

### 점수 기본
| 조건 | 점수 |
|------|------|
| 광 3장 | 3점 |
| 광 4장 | 4점 |
| 광 5장 | 15점 |
| 비광 포함 시 | 광 카운트 -1 (즉 비광 포함 4광은 2점) |
| 열끗 5장 | 1점 (추가 +1점 per 장) |
| 고도리 (2·4·8월 열끗) | 5점 |
| 띠 5장 | 1점 (추가 +1점 per 장) |
| 단(홍/청/초) 3장 모음 | 각 3점 |
| 피 10장 | 1점 (쌍피 = 2장 가치, 추가 +1점 per 장) |

### 보너스 룰 (Phase 4 구현)
- **흔들기**: 같은 월 3장 손패 → 점수 ×2
- **폭탄**: 같은 월 4장 손패 → 점수 ×2 + 상대방 피 1장 빼앗기
- **따닥**: 같은 월 2장 동시 매칭 → 상대방 피 1장 빼앗기
- **쪽**: 손패와 더미가 같은 월일 때
- **뻑**: 같은 월 3장 동시 → 카드 선택 모달
- **자뻑**: 본인이 뻑 카드를 가져옴
- **싹쓸이**: 바닥 모두 가져오기 → 상대방 피 1장씩 빼앗기

### 박 (점수 2배)
- **피박**: 7장 미만 → 상대 점수 ×2
- **광박**: 광 0장 → 상대 점수 ×2
- **멍박**: 열끗 0장 → 상대 점수 ×2
- **고박**: 고를 부른 사람이 점수 못 내고 게임 종료

### 고/스톱
- 7점 도달 시 선택
- **고**: 점수 누적 + 다음 점수 획득 시 추가 배수
- **스톱**: 현재 점수 확정, 게임 종료
- 1고 = ×1, 2고 = ×2, 3고 = ×3 (이후 누적)

## 기능 목록 (Tier별)

### Tier 1: MVP 필수 (Phase 1-3 통합 구현)
1. 친구 초대 링크 + QR 코드 (방 URL 공유)
2. 가능한 카드 하이라이트 (입문자 모드)
3. 점수 자동 계산 + 박 시각화 (분해 표시)
4. 재연결 처리 (네트워크 끊김 복구)
5. 닉네임 + 이모지 아바타 (회원가입 없음)
6. 누적 점수 (여러 판 세션 단위)

### Tier 2: 차별점 기능 (Phase 4-5)
7. **이모지 반응** (영상 위 떠오름) ⭐
8. 말하는 사람 강조 + 음성 파형 (LiveKit 내장)
9. 룰 변형 토글 (시작점수/국준/흔들기/멍따)
10. 관전자 사이드 베팅 (가짜 포인트)
11. **게임 통계 보드** (광부/호구/도박꾼/평화주의자) ⭐

### Tier 3: 여유 있을 때
12. AI 대전 모드 (3단계 난이도)
13. 리플레이 (액션 히스토리 활용)
14. 게임 일시정지 (모두 동의)
15. AFK 자동 처리 (30초 응답 없음)
16. 모바일 화면 꺼짐 방지 (Wake Lock API)
17. 음성 인식 "고!"/"스톱!" 외침

## 에셋 전략

### 임시 (개발 중)
- 인터넷에서 무료/CC0 라이선스 에셋을 검색하여 placeholder로 사용
- 각 작업 단계 도래 시 그때그때 확보
- 저장 위치: `apps/web/public/assets/`
- 출처/라이선스 메모: 같은 폴더에 `LICENSES.md`

**필요 에셋:**
- 화투 카드 이미지 48장 (Phase 1 후반)
- 효과음: 카드 놓는 소리, GO 외침, 점수 획득, 박 발생 (Phase 5)
- 배경 음악 (옵션)
- 이모지 반응용 이미지 (이모티콘 유니코드로도 충분)

### 최종 (정식 배포 시)
- 사용자가 직접 정식 에셋을 구하거나 의뢰하여 교체
- AI 이미지 생성 사용 안 함 (사용자 의향)

## 법적 고려사항

### 한국 게임산업법 (중요)
**고스톱은 사행성 게임으로 분류됩니다.** 상용 운영 시 다음 규제가 적용됩니다:

- **현금/현금성 재화 베팅 절대 금지** (게임산업법 위반)
- **게임물관리위원회 등급분류 필수**
- **웹보드 게임 규제 적용 가능성**:
  - 1일 손실한도 제한
  - 1회 베팅한도 제한
  - 본인인증 의무
  - 24시간 접속 제한

### 본 프로젝트의 처리
- **친구용 비공개 서비스로 한정** → 사행성 규제 회피
- 어떠한 형태의 베팅/환전 기능 미포함
- 점수는 순수 게임 내 가짜 점수로만 존재
- 정식 상용 운영 시 변호사 자문 필수

### 라이선스
- 코드: 개인 프로젝트 (라이선스 미정, 추후 결정)
- 에셋: 각 에셋마다 라이선스 명시 필수 (CC0/MIT 등)
