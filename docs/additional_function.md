  추가로 있으면 좋을 기능을 우선순위별로 정리했습니다.

  🥇 Tier 1 — MVP에 꼭 있으면 좋은 것

  1. 친구 초대 링크

  - 방 URL 복사 (/room/abc123) → 카카오톡으로 한 번에 공유
  - QR 코드 생성 → 모바일에서 빠른 입장
  - 왜? 친구 모이는 마찰 0으로 — 이게 없으면 매번 닉네임 입력하라고 안내해야 함

  2. 가능한 카드 하이라이트 (입문자 모드)

  - 내 차례에 낼 수 있는 카드만 손패에 강조 (테두리 + 미세한 떨림)
  - 어떤 바닥 카드와 매칭되는지도 화살표로 표시
  - 왜? 친구 중 화투 잘 모르는 사람이 있을 가능성 높음. 학습 곡선 부드럽게

  3. 점수 자동 계산 + 박 시각화

  - 게임 끝나면 점수 내역을 풀어서 보여줌:
  광 3장   = 3점
  띠 6장   = 2점
  피 12장  = 3점 (쌍피 1 포함)
  --- 총 8점 ×2 (2고) = 16점
  ⚠️ 상대 1: 광박 ×2 = 32점
  - 왜? 점수 계산 룰이 복잡 → 분쟁 해소 + 학습 효과

  4. 재연결 처리

  - 네트워크 끊겼다 돌아오면 자동으로 게임 복구
  - 5초 이상 끊기면 다른 플레이어에게 "○○ 재접속 중..." 표시
  - 왜? 모바일 와이파이 신호 약하거나 LTE 전환 시 끊김 흔함

  5. 닉네임 + 이모지 아바타

  - 회원가입 없이 닉네임만 입력
  - 이모지 32개 중 선택해서 아바타로 사용 (🐱🐶🦊🐼...)
  - 왜? 가입 마찰 0 + 영상 꺼도 누군지 알 수 있음

  6. 누적 점수 (여러 판)

  - 한 게임이 아니라 "오늘의 세션" 단위로 점수 누적
  - 5판 정도 한 다음 최종 결과 정산
  - 왜? 한 판으로 끝내면 운빨 너무 큼. 여러 판으로 실력 가늠 가능

  ---
  🥈 Tier 2 — 친구용 + 화상채팅에 특히 잘 맞는 것

  7. 이모지 반응 (영상 위 떠오름)

  - 영상 클릭 → 이모지 선택 (😂🔥💀🤡👏)
  - 본인 영상 위에서 위로 떠오르며 사라짐 (3초)
  - 왜? 다른 사람 카드 낼 때 무음으로 반응 가능. 5명 동시 떠들면 정신없음 → 시각 반응으로 대체

  8. 말하는 사람 강조 + 음성 파형

  - 누가 말하고 있는지 영상 외곽선 파란색으로 강조 (LiveKit이 제공)
  - 음성 파형도 작게 표시 (마이크 음소거 여부 확인용)
  - 왜? 5명 영상 띠에서 누가 말하는지 즉시 파악

  9. 룰 변형 토글

  - 방 만들 때 옵션 선택:
    - 시작 점수 (3점 / 7점)
    - 국준 처리 (쌍피 / 광)
    - 흔들기 룰 (2배 / 점수만)
    - 멍따 인정 여부
  - 왜? 친구 그룹마다 룰이 다름. 분쟁 방지

  10. 관전자 사이드 베팅 (가짜 점수)

  - 관전자는 게임 시작 전 "1등 누구일까?" 베팅
  - 맞히면 가짜 포인트 획득 (오늘의 베팅왕)
  - 왜? 관전자가 그냥 보기만 하면 지루함 → 참여감 부여

  11. 게임 통계 보드

  - 세션 끝나면 재미있는 통계:
    - 🏆 오늘의 광부 (광 점수 1등)
    - 💀 오늘의 호구 (박 가장 많이 당한 사람)
    - 🎲 오늘의 도박꾼 (GO 가장 많이 한 사람)
    - 🐢 오늘의 평화주의자 (스톱 가장 많이 한 사람)
  - 왜? 친구끼리 놀릴 거리 만들기 + 스크린샷 공유 욕구 자극

  🥉 Tier 3 — 여유 되면

  12. AI 대전 모드

  - 친구들 모이기 전 혼자 연습 가능
  - AI 난이도 3단계 (초보/중수/고수)
  - 왜? 친구 모이기 전 5분 시간 때우기 + 룰 학습용

  13. 리플레이

  - 액션 히스토리는 이미 저장하므로 자연스럽게 가능
  - 게임 끝나고 "다시보기" 버튼 → 빨리감기 가능
  - 왜? 명장면 (역전 GO 5번 등) 다시 보고 싶을 때

  14. 게임 일시정지

  - "잠깐만!" 버튼 → 모두 동의해야 다시 시작
  - 30초 이상 일시정지 시 자동 재개 (또는 게임 종료)
  - 왜? 화장실 / 전화 / 배달 도착 — 친구놀이의 현실

  15. AFK 자동 처리

  - 30초 이상 응답 없으면 가능한 카드 중 무작위 자동 플레이
  - 3턴 연속 AFK면 봇 처리 또는 강퇴
  - 왜? 5명 중 1명 잠수 타면 게임 못 함

  16. 모바일 화면 꺼짐 방지

  - Wake Lock API로 게임 중 화면 안 꺼지게
  - 왜? 5명 중 1명 잠수 타면 게임 못 함

  16. 모바일 화면 꺼짐 방지

  - Wake Lock API로 게임 중 화면 안 꺼지게
  - 왜? 폰으로 플레이할 때 30초 안 만지면 화면 꺼져서 게임 끊김

  17. 음성 인식 "고!"/"스톱!" (재미요소)

  - 큰 소리로 "고!" 외치면 자동으로 GO 버튼 클릭
  - "스톱!" 외치면 STOP
  - 왜? 한국 고스톱 특유의 외치는 문화 살리기 (옵션 ON/OFF)
